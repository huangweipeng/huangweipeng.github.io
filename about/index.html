关于博客

初中时开始接触到 WordPress，使用免费的空间 000webhost 以及一好心网友提供的二级域名搭建了人生中第一个博客，之后自己注册了个顶级域名，无奈发现名字拼音以 com 和 cn 结尾的都被注册了，无奈只好注册了个 huangweipeng.org 来使用。

无意间了解到 Typecho，对比起 WordPress 的臃肿，它仅只有几百 KB，果断抛弃了 WordPress 选择了 Typecho，主机也随之迁移到了 云左主机 上。

再后来又使用 Hexo 以及 GitHub 搭建了一个博客，写了些文章没怎么维护，直至现在我发现 huangweipeng.cn 到期了被我抢注过来，又重新开始折腾博客，之前博客的内容都没做迁移，干干净净的重新开始仿如新生。
